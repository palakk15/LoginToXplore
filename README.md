 Title of the Project

**Student Enrollment Form using JsonPowerDB**

---

📄 Description

This project is a web-based Student Enrollment Form developed using HTML, JavaScript, Bootstrap, and JsonPowerDB (JPDB). The form allows users to store, update, and retrieve student data efficiently.

The system uses Roll Number as the Primary Key, ensuring that each student record is unique. Based on the entered Roll Number, the system checks whether the record exists in the database. If the record is new, users can save it; if it already exists, users can update it.

The project demonstrates CRUD operations (Create, Read, Update) using JsonPowerDB in a simple and user-friendly interface.

---

 Benefits of using JsonPowerDB

* High Performance: Fast response with minimal latency
* Schema-free: No need to predefine structure like traditional databases
* Easy Integration: Works smoothly with JavaScript and web apps
* Lightweight: No complex setup required
* Real-time Data Processing
* Built-in API Support: No need for backend coding
